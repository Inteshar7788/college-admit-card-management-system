🎓 College Admit Card Management System

A web-based College Admit Card Management System built using PHP, MySQL, Bootstrap, and XAMPP.

This project allows administrators to upload, manage, search, edit, and delete student admit cards while providing students with an easy way to access their admit cards online.

---

🚀 Features

Admin Panel

- Secure Admin Login
- Upload Student Admit Cards (PDF)
- View All Admit Cards
- Search by Student Name or Roll Number
- Edit Student Information
- Delete Admit Cards
- Dashboard Statistics
- Responsive Bootstrap Interface

Student Portal

- Search Admit Card using Roll Number
- View Admit Card Online
- Download Admit Card PDF

⚙️ Installation

Step 1

Clone the repository

git clone https://github.com/Inteshar7788/college-admit-card-management-system.git

Step 2

Move the project folder into:

C:\xampp\htdocs\

Step 3

Start:

- Apache
- MySQL

from XAMPP Control Panel.

Step 4

Create a database:

admitcard_db

Step 5

Import your SQL database file.

Step 6

Update database credentials in:

config/db.php

Step 7

Open:

http://localhost/admitcard
